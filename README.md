## 📖 Описание программы

Movie Library — это графическое приложение для управления личной коллекцией фильмов. Программа позволяет добавлять, удалять, искать и фильтровать фильмы, а также сохраняет все данные в формате JSON.

### Основные возможности

- ✅ Добавление фильмов (название, жанр, год, рейтинг)
- ✅ Удаление выбранных фильмов
- ✅ Фильтрация по жанру и году выпуска
- ✅ Поиск по названию фильма
- ✅ Автоматическое сохранение в JSON
- ✅ Валидация вводимых данных
- ✅ Удобный графический интерфейс на Tkinter
- ✅ Строка состояния с общим количеством фильмов

### Валидация данных

| Поле | Требования |
|------|------------|
| Название | Не может быть пустым |
| Жанр | Не может быть пустым |
| Год выпуска | Число от 1800 до текущего года |
| Рейтинг | Число от 0 до 10 (можно с десятичной точкой) |
